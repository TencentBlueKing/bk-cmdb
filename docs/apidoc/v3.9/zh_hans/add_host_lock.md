### 功能描述

根据主机的id列表对主机加锁，新加主机锁，如果主机已经加过锁，同样提示加锁成功(v3.8.6)

### 请求参数

{{ common_args_desc }}

#### 接口参数

| 字段                |  类型       | 必选   |  描述                            |
|---------------------|-------------|--------|----------------------------------|
|id_list| int array| 是| 主机ID列表|


### 请求参数示例

```python
{
   "id_list":[1, 2, 3]
}
```

{
    "result": true,
    "code": 0,
    "message": "success",
    "data": null
}

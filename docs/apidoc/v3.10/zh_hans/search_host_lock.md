### 功能描述

根据主机id列表查询主机锁(v3.8.6)

### 请求参数

{{ common_args_desc }}

#### 接口参数

| 字段                |  类型       | 必选   |  描述                            |
|---------------------|-------------|--------|----------------------------------|
|id_list| int array| 是|主机ID列表|


### 请求参数示例

```python
{
   "id_list":[1, 2]
}
```

### 返回结果示例

```python

{
    "result": true,
    "code": 0,
    "message": "success",
    "data": {
        1: true,
        2: false
    }
}
```

### 返回结果参数说明

#### data
| 字段      | 类型      | 描述         |
|-----------|-----------|--------------|
| data | map[int]bool | 请求返回的数据, key 是 ID，value 是否上锁 |
